What can this app do?  
 1.View the brief information of the restaurant  
 2.View detailed information by clicking  
 3.Find specific restaurant by name or category  
 4.Sorting restaurant by category  
  
What's the enviorment to run this app?  
 1.node.js v20.18.0  
 2.express v4.21.1  
 3.nodemon v3.1.7  
 4.express-handlebars v8.0.1  
*if you face any problems using the app, it's recommended to use the exactly same version  
  
How to start the app?  
 1.make sure you installed node.js  
 2.use "npm install" to set up the environment  
 3.use "npm run dev" to run the server  
 4.visit the app by the url "localhost:3000"
